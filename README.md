## 🚀 Projetos que já desenvolvi

### 🛣️ [Road Help](https://github.com/GustavoSilva-dev/RoadHelp-Project)
MVP de uma Plataforma de GPS dedicada para veículos de carga com rastreamento e geração automatizada de rotas.  
**Stack:** ```React``` • ```Node.js``` • ```Express.js``` • ```MongoDB``` • ```Prisma ORM``` • ```TomTom Maps API``` • ```OpenStreetMap```

### 🏥 [Voll Med API](https://github.com/GustavoSilva-dev/API-REST-Clinica-Medica-SpringBoot)
API RESTful robusta desenvolvida com Spring Boot para cadastro de médicos e pacientes com gerenciamento e registro de agendamentos, com controle de acesso, documentação, testes unitários e geração de tokens.  
**Stack:** ```Java 21``` • ```Spring Boot``` • ```Spring Security``` • ```Hibernate JPA``` • ```JUnit``` • ```JWT``` • ```Postman API``` • ```Flyway``` • ```Swagger```

### 🎮 [TaskCoin](https://github.com/GustavoSilva-dev/TaskCoin-Project)
Plataforma web de gamificação de tarefas, incentivando crianças a realizar atividades do cotidiano através de um sistema de recompensas, níveis e ofensiva, com controle total dos responsáveis.  
**Stack:** ```React``` • ```Spring Boot``` • ```PostgreSQL``` • ```Supabase``` • ```Insomnia API``` • ```Docker``` • ```Swagger```
